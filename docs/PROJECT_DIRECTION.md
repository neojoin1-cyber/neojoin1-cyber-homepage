# 설탕과소금 프로젝트 방향

## 핵심 철학

설탕과소금은 이 세상에 꼭 필요한 설탕과 소금 같은 인재를 기르고 돕겠다는 개인적 인생관이자 경북직업교육연구회의 중요한 모토이다.

개인 홈페이지는 이 철학을 설명하는 장소이면서, 여러 독립 시스템으로 들어가는 포털 역할을 맡는다. 실제 기능은 한 홈페이지 안에 모두 섞지 않고 별도 프로젝트로 분리해 연결한다.

## 포털의 역할

- 홈 화면은 개인 이력보다 콘텐츠와 시스템 입구를 우선한다.
- 개인 소개, 활동 기록, 교직 이야기는 `주인장` 영역에 모은다.
- 법률정보, 전자책 서재, AI 도구, 기록 보관소는 독립 프로젝트로 확장한다.
- 각 시스템은 필요할 때 홈페이지 카드나 메뉴에서 연결한다.

## AI 법률정보 도우미의 방향

첫 시스템은 특성화고등학교 학생, 선생님, 학부모, 학교 구성원의 취업과 학교 현장 문제를 돕는 법률정보 시스템으로 준비한다.

초기 핵심 주제는 다음과 같다.

- 특성화고 취업 지원
- 도제학교
- 현장실습
- 해외 현장실습
- 중대재해처벌법
- 학교폭력
- 정규 교사와 기간제 교사
- 행정실 정직원과 상근 기간제 행정직원
- 학생관리
- 민원 대응
- 선생님들의 개인적 법률정보 지원

이 시스템은 법률 자문이나 사건 판단이 아니라 공개 법령, 판례, 행정자료, 원문 근거를 찾기 쉽게 정리하는 정보 제공 도구로 표현한다.

## 교육 전자책 서재의 방향

교육강의자료실은 단순 자료 목록이 아니라 전자책 서재 시스템으로 발전시킨다.

- 강의 자료
- 강연 자료
- AI 활용 수업 자료
- 교육자료실
- 교육정보 나눔
- 경북직업교육연구회 활동 자료

사용자가 자료를 모아 읽고, 주제별로 찾고, 나중에는 전자책처럼 열람할 수 있는 독립 시스템으로 설계한다.

## 확장 기준

- 새 기능은 설탕과소금의 철학과 교육 현장 지원 목적에 맞는지 먼저 확인한다.
- 포털은 입구, 독립 프로젝트는 실제 기능이라는 구조를 유지한다.
- 법률정보 시스템에서는 항상 원문 근거와 정보 제공 고지를 우선한다.
- 전자책 시스템은 기존 보안 전자책 프로젝트와 섞지 않고 새 프로젝트로 검토한다.
- Firebase, API 키, 배포, 새 저장소 생성은 실행 전 사용자 확인을 받는다.
